# Changelog

## 1.0.0 (2026-03-24)

Initial release.

- Three-mode workflow: discover, audit, rewrite
- Purpose classification system (Directive / Structured / Generative)
- Tiered prompt discovery (standalone files, inline code, config-embedded)
- Multi-language support: Python, JS/TS, Go, Ruby, Rust, Java/Kotlin
- R/T/S/C/E/N framework reference document (822 words)
- Qualitative grading (Strong / Needs Work / Major Gaps) with per-prompt scorecards
- Rewrite modes: extract, in-place, hybrid
- Deduplication and zero-results handling
