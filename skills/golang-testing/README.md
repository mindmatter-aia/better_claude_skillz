# `/golang-testing` -- Go Testing Patterns

Go testing patterns including table-driven tests, subtests, benchmarks, fuzzing, and test coverage following TDD methodology with idiomatic Go practices.

## Usage

```
/golang-testing
```

## Features

- TDD workflow with RED-GREEN-REFACTOR cycle for Go projects
- Table-driven tests with error cases for comprehensive coverage
- Benchmark patterns for performance measurement with memory allocation tracking
- Fuzz testing (Go 1.18+) for input validation and property-based testing
- Interface-based mocking, golden files, and HTTP handler testing patterns

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r golang-testing ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse golang-testing "$env:USERPROFILE\.claude\skills\"
```
