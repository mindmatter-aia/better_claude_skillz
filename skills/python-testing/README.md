# `/python-testing` -- Python Testing Strategies

Comprehensive testing strategies for Python applications using pytest, TDD methodology, fixtures, mocking, parametrization, and coverage requirements.

## Usage

```
/python-testing
```

## Features

- TDD cycle enforcement: RED, GREEN, REFACTOR
- pytest fundamentals with assertion patterns
- Fixtures: basic, setup/teardown, scopes, parameterized, autouse, conftest
- Parametrization with IDs for readable test output
- Mocking: functions, return values, exceptions, context managers, autospec
- Async testing with pytest-asyncio
- Test organization: directory structure, test classes, markers
- Coverage configuration with 80%+ targets
- Common patterns for API endpoints, database operations, and class methods

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r python-testing ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse python-testing "$env:USERPROFILE\.claude\skills\"
```
