# `/coding-standards` -- Universal Coding Standards

Universal coding standards, best practices, and patterns for TypeScript, JavaScript, React, and Node.js development.

## Usage

```
/coding-standards
```

## Features

- Code quality principles: KISS, DRY, YAGNI, and readability-first approach
- TypeScript/JavaScript naming conventions, immutability patterns, and error handling
- React best practices for components, hooks, state management, and rendering
- API design standards with REST conventions, response formats, and input validation
- Testing standards with AAA pattern and code smell detection

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r coding-standards ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse coding-standards "$env:USERPROFILE\.claude\skills\"
```
