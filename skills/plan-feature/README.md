# `/plan-feature` -- Comprehensive Feature Planning

Create a comprehensive feature plan with deep codebase analysis, external research, and strategic planning for one-pass implementation success.

## Usage

```
/plan-feature Add user authentication with OAuth2
```

## Features

- 5-phase planning process: Understanding, Intelligence Gathering, Research, Strategic Thinking, Plan Generation
- Deep codebase analysis with pattern recognition and dependency mapping
- External documentation research with specific section anchors
- Context-rich plans that pass the "No Prior Knowledge Test"
- Atomic, independently testable tasks with validation commands
- Confidence scoring for one-pass implementation success

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r plan-feature ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse plan-feature "$env:USERPROFILE\.claude\skills\"
```
