# `/golang-patterns` -- Idiomatic Go Development Patterns

Idiomatic Go patterns, best practices, and conventions for building robust, efficient, and maintainable Go applications.

## Usage

```
/golang-patterns
```

## Features

- Core principles: simplicity, zero-value usefulness, and interface-first design
- Error handling patterns with wrapping, custom types, and sentinel errors
- Concurrency patterns: worker pools, context cancellation, errgroup, and leak prevention
- Package organization with standard project layout and naming conventions
- Performance patterns: slice preallocation, sync.Pool, and string builder

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r golang-patterns ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse golang-patterns "$env:USERPROFILE\.claude\skills\"
```
