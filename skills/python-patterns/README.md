# `/python-patterns` -- Pythonic Idioms and Best Practices

Idiomatic Python patterns and best practices for building robust, efficient, and maintainable applications.

## Usage

```
/python-patterns
```

## Features

- PEP 8 standards and type hint patterns (Python 3.9+)
- EAFP error handling with exception chaining
- Context managers for resource management
- Comprehensions, generators, and lazy evaluation
- Data classes and named tuples
- Function, parameterized, and class-based decorators
- Concurrency patterns: threading, multiprocessing, async/await
- Package organization and import conventions
- Memory optimization with __slots__ and generators
- Anti-pattern detection and corrections

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r python-patterns ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse python-patterns "$env:USERPROFILE\.claude\skills\"
```
