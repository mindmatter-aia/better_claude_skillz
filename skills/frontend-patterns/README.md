# `/frontend-patterns` -- Frontend Development Patterns

Frontend development patterns for React, Next.js, state management, performance optimization, and UI best practices.

## Usage

```
/frontend-patterns
```

## Features

- Component patterns: composition, compound components, and render props
- Custom hooks for state management, data fetching, and debouncing
- Context + Reducer pattern for scalable state management
- Performance optimization with memoization, code splitting, and list virtualization
- Accessibility patterns for keyboard navigation and focus management

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r frontend-patterns ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse frontend-patterns "$env:USERPROFILE\.claude\skills\"
```
