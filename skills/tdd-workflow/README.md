# `/tdd-workflow` -- TDD Workflow Patterns

Comprehensive test-driven development workflow with patterns for unit, integration, and E2E tests. Enforces 80%+ coverage.

## Usage

```
/tdd-workflow
```

## Features

- Step-by-step TDD workflow: user journeys, test cases, fail, implement, pass, refactor, verify
- Unit test patterns with Jest/Vitest and Testing Library
- API integration test patterns with Next.js route handlers
- E2E test patterns with Playwright
- Mocking patterns for databases, caches, and AI services
- Test file organization conventions
- Coverage threshold configuration
- Common testing mistakes with correct alternatives
- CI/CD integration examples with GitHub Actions

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r tdd-workflow ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse tdd-workflow "$env:USERPROFILE\.claude\skills\"
```
